# Windows 电脑教程

若设备中已有**同类软件**或**旧版APP**，请先 **断开连接** 并 **卸载** ，避免出现异常情况导致无法正常使用。

第一次使用，请在购买后，**断开**连接，重启APP，或点击 **更新** 按钮。

## 01 下载地址

[点击下载](https://69.63.199.20:50089/d/home/alist_files/client/dd/net/digilink.exe)

:::tip
安装 并 连接后，若无法使用，请**断开**连接，在屏幕右下角托盘，找到APP图标，鼠标右键退出；

再到桌面找到APP图标，鼠标右键，选择：以管理员身份启动；

点击 右上角的**展开按钮**，勾选**虚拟网 模式**，再次**连接**试用，若依然无法使用，勾选**穿透模式**，**断开**连接，重启APP，再次**连接**试用。
:::

:::tip
安装问题处理：

1. 若浏览器提示无法下载，请按照下方 《02 安装》 步骤操作

2. 若首次打开软件出现报错，请尝试以管理员身份运行客户端：在桌面找到 Digilink 图标，鼠标右键选择以管理员身份运行

3. 若仍提示错误，请完全退出软件后，卸载电脑中的同类软件、旧版客户端、当前客户端，重启电脑后重新安装

完全退出软件方法：

在屏幕右下角软件托盘处，找到 Digilink 图标，鼠标右键选择退出。
